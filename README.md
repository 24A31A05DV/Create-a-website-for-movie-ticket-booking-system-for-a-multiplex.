🎬 Movie Ticket Booking System

This project is a simple Movie Ticket Booking System developed using HTML, CSS, and JavaScript.
It allows users to browse movies, select seats, and simulate ticket booking with a payment interface.

---

🌐 Live Website

https://24A31A05DV.github.io/Create-a-website-for-movie-ticket-booking-system-for-a-multiplex/

---

📌 Features

- User Login and Registration
- Movie Selection Page
- Interactive Seat Selection
- Real-time Ticket Price Calculation
- Payment Interface
- Booking Confirmation

---

🛠 Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages (for deployment)
